# Al Noor Attar - Project Overview

**Al Noor Attar (النور للعطور)** is an international luxury Arabic fragrance startup e-commerce experience designed as a Full Stack Developer Internship showcase.

## 1. Project Objective
To build a highly responsive, modern, culturally expressive landing and products platform that achieves a 100/100 standard on:
*   **UI Quality**: High-contrast, custom styled color modes (Gold/Obsidian), elegant serif display typography, and smooth micro-animations.
*   **Code Quality**: Well-commented, semantic HTML, modular CSS styling files, and native, dependency-free JavaScript.
*   **Performance**: Fast asset loads, sub-second translation boots, and zero heavy frameworks dependencies.

## 2. Page Architecture & Structure
The website is split across five responsive pages:

1.  **Home Page (`index.html`)**: Focuses on brand history, customer credentials, best-seller showcases with custom filter elements, and general FAQs.
2.  **Features Page (`features.html`)**: Details raw cold-pressed distillation parameters, sustainable Cambodia Oud logging systems, and custom gift casing options.
3.  **Pricing Page (`pricing.html`)**: Displays 3 distinct purchasing collections (Miniature, Signature, Imperial) and a comprehensive feature comparison grid.
4.  **Contact Page (`contact.html`)**: Hosts our secure Atelier consultation inquiry dashboard alongside a customized vector coordinates map.
5.  **404 Page (`404.html`)**: Elegant, customized fallback route with bilingual redirection anchors.

## 3. Persistent Core State (Theme & Language)
Two core preferences are managed at compile-time and persisted via browser `localStorage`:
*   **Language (`en` / `bn`)**: Supports fully integrated translation of headings, titles, inputs placeholders, and descriptions. Select toggles are synced dynamically.
*   **Visual theme (`light` / `dark`)**: Preserves the visitor's choice between Obsidian dark canvas and Alabaster white paper.
